About Project

To create an Alumni portal module in OpenEduCat ERP for Universities.

Project Structure

packages folder consit of several packages representing feature component of the application and it's maintained through the npm workspace

serverpackage is a node js application which contains the code to deploy the bundle file to the clod

  • Mongodb

Project Flow Chart

Dashboard for Admin

  1. Contains list of All students-We are adding a CRUD Operation for student
  2. For frontEnd, we are using a card to view about the student details and in that card we can have a two buttons for edit and delete
  3. In edit we are using a Modal component, in that we can have a form and we can store it in the database
  4. For backend by using Mvc pattern in the node js we are creating a folder structure and connnecting with mongodb by using moongose package
  5. Creating routes for read,add,update and delete for student - We are creating a Rest Api end points

Annoucement

  1. We are using a form for front end to add an Announcement
  2. Annonucement will be displayed in the dashboard in UI
  3. By using nodemailer, a copy will be sent to the every alumini

Community discusion-User

  1. Creating a group chat- we are implementing the community discussion
  2. Front End should be Group chat user interface
  3. Backend Should contains MVC pattern and by using Socket.io
  4. Event emitter will send data and receive data in real time
  5. We have store data in db for future reference

Personal Messaging feature

  1. Front End should be one on one chat
  2. Same procedure which was implemented in groupchat should be carried out for Backend
